Add recipes feature with pre-built configuration templates

- `recipe list` - Browse available recipes by category
- `recipe show <name>` - Preview recipe configuration
- `recipe apply <name>` - Deploy recipes to Saleor instances
- `recipe export <name>` - Export recipes for customization
- 4 built-in recipes: multi-region, digital-products, click-and-collect, custom-shipping
- JSON output support for automation
