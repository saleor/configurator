Add recipes feature with pre-built YAML configuration templates

- Add `recipe` command to list available recipes by category
- Add `recipe-show` command to preview recipe configuration
- Add `recipe-apply` command to deploy recipes to Saleor instances
- Add `recipe-export` command to export recipes for local customization
- Include 4 built-in recipes: multi-region, digital-products, click-and-collect, custom-shipping
- Support JSON output for automation and AI agent integration
